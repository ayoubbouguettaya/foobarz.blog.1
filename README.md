## Run the Project

feel free to  clone the project make adjustement to play arround with it.

### Requirement

* Git tool
* NodeJs (preferebly > 16)

### Run Backend

```{shell}
cd /backend
//install yarn
yarn
yarn start:dev
```

### Run Front

```{shell}
cd /frontend
//install yarn
yarn
yarn dev
```

> Now Open the Browser on  <http://127.0.0.1:5173/>
